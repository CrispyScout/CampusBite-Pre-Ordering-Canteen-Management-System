# CampusBite-Pre-Ordering-Canteen-Management-System
CampusBite is a full-stack web application that modernizes the food ordering experience in college canteens. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js) with Firebase for authentication and hosting, it eliminates the common problems of long queues, manual billing errors, and order mismanagement during peak hours.
